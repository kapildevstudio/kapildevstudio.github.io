# Kapil Dev Studio

Kapil Dev Studio is an independent digital studio brand focused on building practical products with a code-first mindset.

The current public website introduces the studio, presents its visual identity, and provides support, privacy, and account-deletion information for the studio's published mobile app:

- `SNAKE - Egg Hunter`

## Brand Direction

Kapil Dev Studio is positioned around a simple builder philosophy:

- Code
- Build
- Ship

The broader brand direction supports:

- Mobile games
- Developer-led projects
- Product experiments
- Tutorials and real-world technical work

## Website Pages

The site currently includes:

- `index.html` - studio homepage
- `contact.html` - support and inquiries
- `privacy-policy.html` - privacy details for `SNAKE - Egg Hunter`
- `delete-account.html` - account and local-data deletion guidance

## Featured App

`SNAKE - Egg Hunter` is the current featured release under the Kapil Dev Studio banner. The app support and legal pages are part of this repository so the public site can serve both studio branding and required app information.
